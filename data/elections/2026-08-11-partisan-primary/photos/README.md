# Candidate photos

Headshots referenced by the `photo` field in ballot.json. Conventions:

- File name: `<candidate-id>.jpg` (e.g. `clark-fred.jpg`), square crop,
  at least 112×112.
- **Source rule (same as bios): candidate-supplied only.** Use headshots
  the campaign provides or explicitly publishes for press use. No news
  outlet photos, no social media grabs — rights and verification both.
- A candidate without a photo automatically gets a neutral initials
  placeholder in the same frame. Never substitute a stock image, and do
  not withhold other candidates' photos to even things out — the
  placeholder is the designed equal treatment.
- Set the ballot.json `photo` field to the file name to activate it.
