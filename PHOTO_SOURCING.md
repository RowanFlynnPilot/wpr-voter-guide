# Candidate Photo Sourcing Worklist

**Status: UNVERIFIED research — nothing here is approved.** No image ships
until a human fills in the Decision column for its row. Compiled July 4,
2026 from Wikimedia Commons, official government pages, campaign sites,
and PBS Wisconsin candidate profiles. No images have been downloaded.

How this interacts with the rules: CLAUDE.md editorial rule 7 currently
says candidate-supplied headshots only. Approving any non-campaign source
below (Commons, official portrait, PBS Wisconsin) is an editorial policy
decision that loosens rule 7 — decide once, then apply per row. Whatever
is approved: file goes in `data/elections/2026-08-11-partisan-primary/photos/`
as `<candidate-id>.jpg` and the candidate's `photo` field is set. If any
approved license requires visible attribution (the CC BY files do), say so
in the Decision column — the card UI needs a small photo-credit line added
before those ship (not built yet; one small code change when first needed).

Decision column: leave blank = undecided · `USE` (+ credit text if needed)
· `ASK CAMPAIGN` · `SKIP` (placeholder stays).

## Cautions on specific Commons files

- **Joel Brennan** ([File:Joel_Brennan.jpg](https://commons.wikimedia.org/wiki/File:Joel_Brennan.jpg)):
  CC0 "own work" uploaded by a pseudonymous account the day he announced;
  looks like a campaign headshot. Likely license-washed — do not use
  without independent confirmation or campaign permission.
- **David Crowley**: public-domain claim rests on a photo from a senator's
  tweet — weak provenance for a PD assertion.
- **Kelda Roys**' only Commons file is from 2008 — too dated; use the
  Senate page instead.
- **Sarah Godlewski**'s CC BY-SA Commons file is 234×223px — below print
  quality.
- CC0/own-work files (Hong, Rodriguez) carry the standard Commons risk
  that the uploader's claim is wrong; both currently look sound (used as
  Wikipedia infobox images, descriptions match).
